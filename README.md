# vue-template-pc

## 文件目录结构
+ src
   + api               --- 集中化接口管理
   + assets            --- 静态资源
   + components        --- 公共组件
   + http              --- axios二次封装和全局请求拦截
   + router            --- 路由管理
   + style             --- 全局样式
   + view              --- 业务页面
   + vue.config.js     --- webpack配置
   + prod.env.js       --- 动态打包环境配置
